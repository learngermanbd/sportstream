What Is Sportzfy?
A live sports streaming app for Android

Sportzfy (package: com.blaze.sportzfy) is a live sports streaming Android application at version 22.0 (versionCode 31). It streams live sports events, TV channels, and match highlights. The app targets Android SDK 36 with a minimum SDK of 23 (Android 6.0 Marshmallow), making it compatible with a wide range of devices including phones, tablets, and Android TV.

Video Player Deep Dive
Custom Media3/ExoPlayer with advanced controls
Play/Pause Rewind Fast Forward Seek Bar + Buffer Screen Lock/Unlock Picture-in-Picture Resize/Fit Video Quality Subtitles Volume Control Swipe Gestures Link Switching Stream Update Banner Refresh Button Share Button Title Display

The player features a custom controller layout with a top bar (back, title, lock, PiP, resize, settings), center controls (rewind, play/pause, fast forward), and bottom bar (time/seek bar, volume, fullscreen). It supports multiple stream sources per event with a horizontal RecyclerView for link selection, and displays a notification banner when stream links change. Swipe gestures enable brightness and volume adjustment.

Protection & Security
Native code protection and obfuscation

The app uses native code protection via libabcd.so / libabcdProtect with multiple layers:

🔒 String Decryption — XOR cipher with key 16 for all sensitive strings
🔐 File Integrity Checks — Byte-by-byte comparison of asset files
🔑 Reflection-based Verification — Runtime class/method validation
📦 DEX Protection — Encrypted assets (libabcd.enc, libabcd2.enc)
🧬 Heavy Obfuscation — All class/method names are CJK characters or single letters
